# Insurance Analytics Project for AlphaCare Insurance Solutions

This repository contains the code and analysis for optimizing car insurance marketing using historical claim data.

## Project Structure
- data/ : For data files
- notebooks/ : For Jupyter notebooks
- scripts/ : For Python scripts
- reports/ : For reports and plots

## Setup
- Clone the repo: `git clone <your-repo-url>`
- Install dependencies: `pip install -r requirements.txt`

## Tasks
- Task 1: EDA and Stats
- Task 2: DVC
- Task 3: Hypothesis Testing
- Task 4: Modeling
